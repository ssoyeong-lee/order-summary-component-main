### What I learned

블럭의 정렬 방법
1. position: absolute; 선언
2. top 또는 left 속성을 사용하여 원하는 위치 설정
3. margin-top 또는 margin-left 속성을 이용, 블럭의 width와 height 값의 반 픽셀만큼 위치 이동 

```css
.block {
    width: 50px; height: 50px;
    position: absolute;
    top: 50%;
    margin-top: -25px; 
}
```
